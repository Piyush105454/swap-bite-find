# Swap Eat

Swap Eat is a platform that allows people and businesses to share surplus food instead of wasting it. Users can post food items, view available items on a map, and chat in real-time with other users.

## Project Info

**Live URL**: [https://swap-bite-find.onrender.com/](https://swap-bite-find.onrender.com/)
